# MathBridge AI: Final project – AI application design and development.

MathBridge AI is an educational AI chatbot designed to help middle school students (ages 12–14) understand math through 
simple, real-world analogies.


## Project Overview

Many students struggle with math because it feels abstract and difficult. MathBridge AI solves this by turning math concepts 
into relatable everyday situations, making learning easier and more engaging.

Example:
Instead of explaining quadratic equations with formulas, the AI compares them to the path of a thrown ball.


## Features

- Simple, real-world explanations
- Age-appropriate language (12–14 years old)
- Interactive learning with follow-up questions
- AI-powered chatbot interface
- Safe and responsible AI design

## How It Works

1. User enters a math topic  
2. AI generates:
   - A real-world analogy  
   - A simple explanation  
   - A math connection  
   - A quick question  
3. Student interacts and learns step by step
